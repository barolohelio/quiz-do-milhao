## Começando....

Clone o repositório no local desejado

* Instale o pacote de dependências. 
   ```bash
    npm install
    # ou 
    yarn install
  ```
  
* Execute o projeto como desenvolvedor
  ```bash
    npm run dev
    # ou 
    yarn run
  ```
  
 Acesse http://localhost:3000 com seu navegador e veja o resultado.
